Migration Guide for 0.6

This documents covers code changes that need to be made by Sapphire adopters as part of migrating to 0.6 release. Only changes from the previous release are covered.

SapphireDiagramEditor

Before After 
page.setDiagramHeaderText( text )

No direct equivalent has been provided, but page header text can now be specified as an expression. This accounts for the dynamic header text use cases. For further guidance on this migration, post a question on Sapphire Adopter Forum.
